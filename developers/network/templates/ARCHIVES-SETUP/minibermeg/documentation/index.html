For a simple two column layout, create a .row and add the appropriate number of .col-lg-* columns. As this is a 12-column grid, each .col-lg-* spans a number of those 12 columns, and should always add up to 12 for each row (or the number of columns in the parent).

JavaScript

In folder js you will find 5 javascript files, but all adjustments are in just one file: "script.js".

All JavaScript files:

  • MinberiMAG plugins js files - modernmag-plugins.min.js
  • Bootstrap - bootstrap.min.js
  • Google Map v4.1 - gmap3.min.js
  • popper - popper.js
  • Jquery code added here - script.js

Description:

  • ("modernmag-plugins.min.js") all third party plugins js code are minified and compressed for better performance in one file.
  • Bootstrap part Bootstrap Framework.
  • JS for Google Map in Contact section. Google Map data enter in "script.js".
  • ("popper.js") is used for bootstrap. Don't remove this file cause bootstrap js code will not work properly.
  • Most of the settings you can adjust in "script.js". Every part is commented so you can have a clue for what is used.
