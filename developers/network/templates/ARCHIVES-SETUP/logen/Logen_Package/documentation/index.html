How to customize Logen easily #back to top

Custom CSS Options - #back to top

We have add a custom style file in customc.css, and it's easy to apply your customize css code that can be done by add to this file

Color Options - #back to top

It's super easy and fast to change the main color of complete site. we have created a separate css file called color.css which has all css. just open that file and replace your color code with existing one that's all, in result you will have you required color scheme on site

Font Options - #back to top

It's super easy and fast to change Font of complete site. we have created a separate css file called fonts.css which has all css. just open that file and replace your Font family with existing one that's all, in result you will have you required Font family on site

Responsive Options - #back to top

In this template we have created a separate css file which has all Responsive style. it's super easy and fast to change any responsive in responsive.css file.
