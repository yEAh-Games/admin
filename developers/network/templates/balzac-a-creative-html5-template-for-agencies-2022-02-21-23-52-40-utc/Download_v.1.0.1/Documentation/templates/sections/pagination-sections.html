Pagination Sections

Pagination

Pagination sections can be added to projects, posts or to the blog index. Each of our templates come with four predesigned pagination sections that are defined by the class .pagination-X, where X represents the pagination section type (1-4) as shown below.

Example
<div class="section-block pagination-X">
	<div class="row">
		<div class="column width-12">
		...
		</div>
	</div>
</div>
CSS For Retina Displays

For pagination sections that have a background image we serve a higher resolution background image by using the media queries below. Note: The CSS targeting below is just an example, refer to each individual feature section for exact CSS targeting.

.pagination-X{
	background-image:url(../images/slider/pagination-bkg-image.jpg);
}

@media
only screen and (-webkit-min-device-pixel-ratio: 2),
only screen and (		min--moz-device-pixel-ratio: 2),
only screen and (			-o-min-device-pixel-ratio: 2/1),
only screen and (				min-device-pixel-ratio: 2),
only screen and (					min-resolution: 192dpi),
only screen and (						min-resolution: 2dppx) { 
		.pagination-X{
			background-image:url(../images/slider/pagination-bkg-image@2x.jpg);
		}
}
Pagination Section One

A simple pagination section for with icons. Target .pagination-1 to set background color or image. Alternatively, for background use Timber background color classes.

Example
<div class="section-block pagination-1 bkg-grey-ultralight">
	<div class="row">
		<div class="column width-6">
			<a href="#" class="pagination-previous">
				<span class="icon-left-circled"></span>
				<small>Previous</small>
				<span>Title Previous..</span>
			</a>
		</div>
		<div class="column width-6">
			<a href="#" class="pagination-next">
				<span class="icon-right-circled"></span>
				<small>Next</small>
				<span>Title Next..</span>
			</a>
		</div>
	</div>
</div>
Pagination Section Two

A full width pagination section. Target the .pagination-2 class to set background color or alternatively use Timber background color classes. Target .column.prev and .column.next respectively for adding background images.

Example
<div class="section-block pagination-2 bkg-grey-ultralight">
	<div class="row full-width collapse">
		<div class="column width-6 prev">
			<a href="#" class="pagination-previous">
				<span class="icon-left-circled"></span>
				<small>Previous</small>
				<span>Title Previous..</span>
			</a>
		</div>
		<div class="column width-6 next">
			<a href="#" class="pagination-next">
				<span class="icon-right-circled"></span>
				<small>Next</small>
				<span>Title Next..</span>
			</a>
		</div>
	</div>
</div>
Pagination Section Three

Traditional pagination with left/right arrows and page numbers. Target the .pagination-3 class to set background color or image. Alternatively, for background use Timber background color classes.

Example
<div class="section-block pagination-3 bkg-grey-ultralight">
	<div class="row">
		<div class="column width-12">
			<ul>
				<li><a href="#" class="pagination-previous icon-left-open"></a></li>
				<li><a href="#" class="current">1</a></li>
				<li><a href="#">2</a></li>
				<li><a href="#">3</a></li>
				<li><a href="#">4</a></li>
				<li><a href="#" class="pagination-next icon-right-open"></a></li>
			</ul>
		</div>
	</div>
</div>
Pagination Section Four

Pagination with buttons. Target the .pagination-4 class to set background color or image. Alternatively, for background use Timber background color classes.

Example
<div class="section-block pagination-4 bkg-grey-ultralight">
	<div class="row">
		<div class="column width-12">
			<a href="#" class="button large pagination-previous bkg-charcoal color-white rounded">Prev Project</a>
			<a href="#" class="button large bkg-charcoal color-white rounded back-to-index"><span class="icon-layout"</span>			</a>
			<a href="#" class="button large pagination-next bkg-charcoal color-white rounded">Next Project</a>
		</div>
	</div>
</div>
Pagination Section Five New

Pagination with buttons. Target the .pagination-4 class to set background color or image. Alternatively, for background use Timber background color classes.

Example
<div class="section-block pagination-5">
	<div class="row">
		<div class="column width-12">
			<ul>
				<li><a href="#" class="pagination-previous"><span class="icon-left-open-mini"></span><span>Back</span></a></li>
				<li><a href="#" class="current">1</a></li>
				<li><a href="#">2</a></li>
				<li><a href="#">3</a></li>
				<li><a href="#">4</a></li>
				<li><a href="#" class="pagination-next"><span>Next</span><span class="icon-right-open-mini"></span></a></li>
			</ul>
		</div>
	</div>
</div>
